# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly via [GitHub Private Vulnerability Reporting](https://github.com/UltraEmbeddedLab/openai-agent-sdk-rust/security/advisories/new).

Please **do not** open a public issue for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
